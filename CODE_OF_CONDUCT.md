# Code of Conduct

## Our pledge
We want this project to be welcoming and respectful for everyone.

## Expected behavior
- Be respectful and constructive.
- Assume good intent.
- Focus feedback on code and behavior, not people.

## Unacceptable behavior
- Harassment, threats, hate speech, or personal attacks.
- Repeated disruptive behavior in issues or PRs.

## Enforcement
Project maintainers may remove comments, close discussions, or block users for severe or repeated violations.
